# Badge Theme Prompt (Future Use)

Use this prompt with Consumer Copilot (or similar image generation tools) whenever we need to create or refresh achievement badge sets.

## Prompt

Create a cohesive set of achievement badge icons for the **VS Code Learn** extension.

### Art direction
- Style: flat/vector, modern, playful, polished
- Visual consistency across the full set (same stroke weight, lighting, corner style, and color treatment)
- High contrast and readable at small sizes
- Transparent background
- No text inside icons
- Centered composition
- Works on both dark and light UI themes

### Output requirements
- Deliver each badge as an individual PNG
- Size: 1024x1024
- Keep safe padding around edges so badges do not look cramped
- Keep all badges in the same theme family

### Theme guidance
Create a distinct but coordinated palette (e.g., blue/teal/purple/gold/coral accents) and keep saturation lively for gamification.

### Badge concepts
1. First Steps — first milestone, launch energy
2. Course Starter — exploration/discovery
3. Course Finisher — graduation/completion
4. Agent Foundations Master — stability/foundations
5. Customization Master — creativity/tuning
6. Agent Extensions Master — integration/power-user
7. Completionist — ultimate all-done achievement
8. Momentum — speed/forward progress
9. Streak Builder — consistency over time
10. Comeback — return after a break

### Naming convention
Return files using this naming pattern:
- `01-first-steps.png`
- `02-course-starter.png`
- `03-course-finisher.png`
- `04-agent-foundations-master.png`
- `05-customization-master.png`
- `06-agent-extensions-master.png`
- `07-completionist.png`
- `08-momentum.png`
- `09-streak-builder.png`
- `10-comeback.png`
